# custom-react-app
해당 저장소는 webpack과 babel을 이용하여 CRA없이 react-app을 구성해보는 연습을 하기 위한 곳입니다.

<br />

### 사용된 webpack plugins
- BannerPlugin
- CleanPlugin
- HtmlWebpackPlugin
- MiniCssExtractPlugin
- OptimizeCssAssetsPlugin
- TerserPlugin

<br />

### 사용된 babel presets
- preset-env
- preset-typescript
- preset-react
